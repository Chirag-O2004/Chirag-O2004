## 🚀 Projects

### 🔬 [HalluciSense](https://github.com/Chirag-O2004) — LLM Hallucination Detection Framework
> *Research-level system targeting publication in Elsevier / Frontiers in AI*

A hybrid framework to detect and quantify hallucinations in LLM outputs with **~94% detection accuracy**.
- **RAG-based fact verification** cross-checks outputs against a grounded knowledge base
- **Confidence scoring** surfaces model uncertainty at the token level
- **Cross-consistency validation** tests semantic stability across rephrased prompts
- Produces a unified **H-Score** — a single hallucination severity metric
- `PyTorch` `HuggingFace` `FastAPI` `RAG` `NLP`

---

### 🚦 [GreenWave AI](https://greenwave-ai.vercel.app) — Emergency Vehicle Preemption System
> *Live demo deployed · Conceptualized for BBMP urban traffic integration*

AI system that clears traffic signals in real time for emergency vehicles.
- **Traffic-aware Dijkstra routing** computes optimal corridor paths dynamically
- **Real-time WebSocket architecture** propagates signal changes with sub-second latency
- Reduces intersection delay by ~**500ms per signal**, directly improving emergency response
- `Next.js` `Node.js` `WebSockets` `Graph Algorithms`

---

### 🐄 [CattleGuard](https://github.com/Chirag-O2004) — AI Livestock Health Monitoring
Edge AI system for continuous cattle health monitoring using simulated IoT sensor data.
- Offline-first architecture — operates without consistent internet connectivity
- Anomaly detection flags health deviations in real time
- `Flask` `React` `TensorFlow Lite` `SQLite` `Docker`
